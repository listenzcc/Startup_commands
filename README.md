## Powershell

### Install posh-git and oh-my-posh

> `Install-Module posh-git`  
> `Install-Module oh-my-posh`

### Setup oh-my-posh theme

> `Notepad $profile`  
> Put `'Set-Theme Honukai'` into `$profile`
